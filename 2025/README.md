# [Advent Of Code](https://adventofcode.com/)

Solutions written in TypeScript by [@MarekE](https://github.com/marek-e)

No AI was used to solve the problems.

## Setup

Install dependencies

```bash
npm install
```

## Running Solutions

```bash
# Run any day/part combination
./run.sh <day> <part>

# Examples:
./run.sh 1 1    # Day 1, Part 1
./run.sh 01 2   # Day 1, Part 2
./run.sh 12 1   # Day 12, Part 1

# Or use npm
npm run solve 1 1
```

## Creating a New Day

```bash
# Create a new day folder with boilerplate files
./new-day.sh <day>

# Example: Create Day02 with part1.ts, part2.ts, and input.txt
./new-day.sh 2
```
